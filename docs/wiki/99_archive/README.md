# Archive

Note ở đây từng đúng nhưng đã bị thay thế (`status: superseded` trong
frontmatter, có `superseded_by` trỏ tới note thay thế). Giữ lại để biết
**tại sao** đã đổi — không xoá lịch sử (nguyên lý "phá vỡ là một quy
trình", xem [[../00_home]]).

Hiện chưa có note nào bị chuyển vào đây — dự án còn mới. Khi 1 note ở
`01_architecture/`, `02_decisions/`, `04_operations/` bị thay thế, chuyển
file đó vào đây nguyên trạng (đổi `status`, thêm `superseded_by`), đừng xoá.
